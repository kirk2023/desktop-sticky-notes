# v1.0.0 首发版本

## 新功能
- 📋 看板视图 - 拖拽式管理事项
- 📝 事项列表 - 支持批量操作
- ⏱️ 计时器 - 精确计时记录
- 📊 甘特图 - 时间线可视化
- 📌 桌面卡片 - 悬浮便签实时查看
- 🗂 归档分析 - 统计完成率、准时率
- 🐱 休息提醒 - 定时提醒休息

## 运行要求
- Python 3.10+
- Windows系统

## 使用方法
1. 解压文件
2. 双击 `desktop_sticky_notes.exe` 运行
3. 首次运行会自动创建数据库
