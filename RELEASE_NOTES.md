## v1.0.0 (2026-04-26) 首个正式版本

### 功能模块
- 事项管理：新建/编辑/删除/归档事项，支持优先级、颜色标签、计划时间通知
- 计时系统：开始/暂停/完成计时，迟到补偿，实际开始时间记录
- 看板系统：多看板管理，自定义甬道，卡片拖拽，事件导入
- 归档分析：时间差异统计，准时率分析，完成日期编辑，归档备注
- 桌面便利贴：悬浮卡片显示，计时状态同步，拖拽移动
- 系统托盘：最小化到托盘，通知提醒，快捷操作

### 运行要求
- Python 3.10+
- Windows 系统

### 使用方法
1. 下载 `desktop_sticky_notes.exe`
2. 双击运行即可（首次启动会自动创建数据库）

或从源码运行：
```bash
pip install -r requirements.txt
python main.py
```
