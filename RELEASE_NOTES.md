# 版本历史

---

## v1.5 (2026-05-23)

### 新功能
- 归档详情支持编辑**实际开始时间**和**实际时长**（小时）
- 看板列表中事项可以编辑实际开始时间

### 改进
- 看板过滤下拉框在看板创建/重命名/删除后自动刷新
- 看板过滤下拉框宽度自适应内容，不再截断长名称
- 切换到事项列表标签时自动刷新看板过滤列表
- 全局 UI 配色优化：深蓝灰色系标题栏，统一按钮配色
- 所有 emoji 替换为文字标签，界面更专业
- 归档详情对话框高度增加，避免内容截断

### 数据库变更
- v11 → v12：events 表新增 `actual_duration_minutes` 字段，支持手动记录实际时长

## v1.0.0 首发版本

### 新功能
- 📋 看板视图 - 拖拽式管理事项
- 📝 事项列表 - 支持批量操作
- ⏱️ 计时器 - 精确计时记录
- 📊 甘特图 - 时间线可视化
- 📌 桌面卡片 - 悬浮便签实时查看
- 🗂 归档分析 - 统计完成率、准时率
- 🐱 休息提醒 - 定时提醒休息

### 运行要求
- Python 3.10+
- Windows 系统

### 使用方法
1. 解压文件
2. 双击 `desktop_sticky_notes.exe` 运行
3. 首次运行会自动创建数据库

下载 EXE 文件：[desktop_sticky_notes.exe](./output/desktop_sticky_notes.exe)
