# 桌面便利贴 - 计划计时系统

> 个人离线效率工具，替代 Jira 进行轻量级项目管理和时间追踪。

## 功能特性

### 事项管理
- 新建/编辑/删除/归档事项
- 支持优先级（高/中/低）、颜色标签
- 计划时间设置与通知提醒
- 事项列表按看板筛选

### 计时系统
- 一键开始/暂停/完成计时
- 迟到补偿机制（自动检测计划时间与当前时间差）
- 实际开始时间记录
- 实际时长自动统计

### 看板系统
- 多看板管理（项目管理/研发流程/个人GTD/自定义模板）
- 看板列表页，双击进入具体看板
- 自定义甬道（添加/删除/重命名）
- 卡片拖拽切换甬道
- 看板内直接新建事件或从事项列表导入
- **Ctrl+滚轮缩放**（甬道宽度、字体、间距同步缩放）
- **卡片右键菜单**（编辑事项 / Pin到桌面）
- **智能甬道匹配**（完成事件后自动移到语义匹配的甬道，如"已上线""已完成"）
- 归属看板为"无"的事件不会出现在任何看板中

### 归档分析
- 时间差异统计（计划时长 vs 实际时长）
- 准时率分析（±5分钟容差）
- 双击弹出详情卡片
- 支持修改完成日期，自动重新计算实际时长
- 归档备注（记录无疾而终、需求取消等原因）
- 归属看板、实际开始时间、完成日期等完整字段

### 桌面便利贴
- 悬浮卡片显示当前事项
- 计时状态实时同步
- 支持拖拽移动和边缘缩放
- 📌 置顶/取消置顶切换
- Pin 切换后自动保持尺寸和位置

### 系统托盘
- 最小化到系统托盘
- 计划时间到达时弹窗通知提醒
- 托盘菜单快速操作

## 技术栈

- **语言**: Python 3.10+
- **GUI 框架**: PyQt5
- **数据库**: SQLite3
- **打包**: PyInstaller

## 快速开始

### 环境要求

- Python 3.10 或更高版本
- Windows 10/11

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行

```bash
python main.py
```

### 打包为 EXE

```bash
python build_exe.py
```

打包完成后自动启动 EXE，可执行文件位于 `output/desktop_sticky_notes.exe`。

## 数据库

应用首次启动时自动创建 SQLite 数据库文件，所有数据存储在本地。

### 主要数据表

| 表名 | 说明 |
|------|------|
| `events` | 事项（标题/描述/时间/优先级/状态/看板归属/实际开始时间等） |
| `timer_records` | 计时记录（开始/暂停/时长） |
| `kanban_boards` | 看板（名称/排序） |
| `kanban_lanes` | 甬道（名称/颜色/所属看板） |
| `kanban_lane_items` | 甬道-事件关联 |
| `card_positions` | 桌面卡片位置和尺寸 |
